# Retail Sales Management System

Simple Python command-line program for a university Software Testing and Quality Assurance lab.

Files:
- `main.py` - command-line interface
- `inventory.py` - inventory functions
- `sales.py` - sales functions
- `utils.py` - small helpers
- `tests/` - pytest tests

Requirements:
- Python 3.8+ (uses only standard library)
- `pytest` to run tests

Run the program:

```bash
python main.py
```

Run tests:

```bash
pytest -q
```

The project is intentionally small and clear for teaching and testing purposes.
